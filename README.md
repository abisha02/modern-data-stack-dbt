# Modern Data Stack with dbt

## Overview

This project demonstrates a production-style ELT pipeline built using PostgreSQL and dbt Core.

The project simulates an e-commerce company where raw transactional data is transformed into analytics-ready models.

## Tech Stack

- PostgreSQL
- dbt Core
- SQL
- Git
- GitHub

## Project Architecture

Raw Data
↓
Staging Models
↓
Intermediate Models
↓
Mart Models
↓
Business Reporting
